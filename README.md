## Descrição do Projeto

Este projeto tem como objetivo analisar o impacto da temperatura e da salinidade da água no tempo de dissolução de comprimidos efervescentes de Vitamina C. Considerando que a vitamina C é hidrossolúvel, a completa dissolução do comprimido é fundamental para garantir a eficácia da suplementação e a adequada absorção dos nutrientes pelo organismo.
